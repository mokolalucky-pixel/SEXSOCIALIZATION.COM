## 💞 What Is This Platform?

**SEXSOCIALIZATION.COM** is a relationship-first social media platform built exclusively for **committed couples and relationship communities**. It is not a dating app — it is a space for people already in relationships to deepen their connection, especially across long distances.

### 🔑 The Core Rule
> **Partners must join together.** You register, invite your partner, and only after your partner accepts can you begin interacting. No solo browsing. No strangers. Just intentional, partner-verified connection.

## 🔵 The Four Circles

The platform is structured around **four distinct social circles**, each with a unique purpose:

| Circle | Who It's For | Purpose |
|---|---|---|
| 👩 **Women's Circle** | Women only | A safe space for relationship advice, support, and sisterhood |
| 👨 **Men's Circle** | Men only | Men supporting men on partnership, love, and commitment topics |
| 👫 **Mixed Circle** | All members | Couples and individuals learning from each other together |
| 🔒 **Private Circle** | You & your partner only | Your exclusive, intimate space — just the two of you |

> Circle access is **gender-verified and server-enforced** to maintain the integrity of each space.

## 🛠️ Technical Architecture

### Core Features

1. **🔐 Authentication** — Verify users via phone number or email
2. **⚡ Realtime Database / Firestore** — Instant message delivery between partners
3. **☁️ Cloud Functions** — Enforce gender-based circle access on the server side
4. **🌐 Hosting** — Live on a real domain: [sexsocialization.com](https://sexsocialization.com)
5. **🗄️ Storage** — Profile pictures and shared media between partners

### Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React.js / Next.js |
| Backend | Node.js + Express |
| Real-time Messaging | Firebase Realtime Database / Firestore |
| Video Calling | WebRTC |
| Auth | Firebase Authentication (Phone / Email / OAuth) |
| Cloud Logic | Firebase Cloud Functions |
| Storage | Firebase Storage / AWS S3 |
| Hosting | Firebase Hosting / Vercel |

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- Firebase CLI
- A `.env` file (see `.env.example`)

### Installation

```bash
# Clone the repository
git clone https://github.com/mokolalucky-pixel/SEXSOCIALIZATION.COM.git
cd SEXSOCIALIZATION.COM

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Start the development server
npm run dev
```

The app will be available at `http://localhost:3000`.

## 🔐 Privacy, Safety & Age Restrictions

- **🔞 Age Restriction: 18+ ONLY.** All users must verify they are 18 years of age or older upon registration.
- All content is shared only between **verified, consenting partners**
- Gender-based circles are **server-enforced** via Cloud Functions
- Users can **revoke partner access** and delete all data at any time
- No content is stored beyond session unless explicitly saved
- Compliant with **GDPR** and **CCPA** privacy regulations
- Zero tolerance for misuse — accounts are subject to review and suspension
