# Client App

Frontend for SEXSOCIALIZATION.COM built with Vite + React.

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Environment

Copy `.env.example` to `.env` and set the needed values.

## Notes

- Routing is handled with `react-router-dom`.
- Authentication is currently a local scaffold (`localStorage`) and should be replaced by a real backend provider.
